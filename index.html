Todo App

    Instructions

    For this assignment, you will need to create a todo list application using the HTML above.

    Setup

    The markup for the todo app is as follows:

    <div class="todo">
  <form id="add-todo">
    <input type="text" placeholder="What do you have to do?" />
    <button>Add</button>
  </form>
  <ul id="todo-list"></ul>
</div>

    Do NOT modify this index.html file. The included automated tests depend on this markup to run. All markup needed to complete this assignment is already included. There is also no need to write any CSS to complete this assignment; all required CSS for the assignment is already included.

    Application Behavior

    When the page loads, the list and input field should be empty. The app should appear as it does in figure 1 below.

    Figure 1

      After typing a todo item into the input field and either pressing the "enter" key OR clicking the "Add" button, the typed item should appear below the form inside of the ul#todo-list element. Because the markup uses a form to capture new list items, this functionality must be attached to a form submit event using JavaScript. See figure 2 and figure 3 for examples of before and after adding a new todo item.

      Figure 2: Typing into the field
        Figure 3: After submitting the form 
        <div class="todo">
  <form id="add-todo">
    <input type="text" placeholder="What do you have to do?" />
    <button>Add</button>
  </form>
  <ul id="todo-list">
    <li>
      <button>
        learn JavaScript
      </button>
    </li>
  </ul>
</div>

        When adding the todo item to the list, that item MUST be a button element nested within a li element as depicted in the above markup. The reason for nesting the button within an li is that it allows the form to be completely keyboard accessible.

        After adding an item, that item may be clicked in the todo list to mark it as done. The item should be marked done by striking through the words using the text-decoration CSS property. See figure 4 for an example.

        Figure 4: Marking an item as done 
        <div class="todo">

  </form>
  <ul id="todo-list">
    <li>
      <button style="text-decoration: line-through">
        learn JavaScript
      </button>
    </li>
  </ul>
</div>

        Additionally, clicking an item that's been marked as done should remove the item from the list (figure 5).
